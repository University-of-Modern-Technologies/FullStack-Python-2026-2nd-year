# FullStack веб-розробка на Python

Курс розрахований на 15 живих занять, що охоплюють 15 основних тем, з 6 домашніми завданнями та фінальним оцінюванням.

## Огляд дисципліни:

- **Тиждень 1:** Тема 1. Базова інженерна інфраструктура Python-проєкту
- **Тиждень 2:** Тема 2. Основи технології Docker
- **Тиждень 3:** Тема 3. Основи Web. Побудова простого HTTP сервера. (Домашнє завдання №1)
- **Тиждень 4:** Тема 4. Асинхронне програмування в Python (Домашнє завдання №2)
- **Тиждень 5:** Тема 5. Основи роботи з ORM SQLAlchemy
- **Тиждень 6:** Тема 6. Розширені можливості SQLAlchemy та міграції схеми баз даних (Домашнє завдання №3)
- **Тиждень 7:** Тема 7. Основи FastAPI
- **Тиждень 8:** Вільний
- **Тиждень 9:** Тема 8. Побудова REST API. Архітектура застосунка (Домашнє завдання №4)
- **Тиждень 10:** Тема 9. Авторизація та Автентифікація
- **Тиждень 11:** Тема 10. Інфраструктурні механізми безпеки та продуктивності backend-застосунку
- **Тиждень 12:** Тема 11. Інтеграція поштових сервісів і cloud-сховища у FastAPI (Домашнє завдання №5)
- **Тиждень 13:** Тема 12. Основи тестування веб-застосунків
- **Тиждень 14:** Тема 13. Практичне заняття. Інтеграційне тестування та документування коду (Домашнє завдання №6)
- **Тиждень 15:** Тема 14. Практичне заняття. Асинхронна обробка та черги повідомлень
- **Тиждень 16:** Тема 15. Практичне заняття. Розгортання та запуск backend-застосунку (Фінальне оцінювання)
- **Тиждень 17-18:** Ментор перевіряє всі домашні завдання, фінальне оцінювання та підраховує підсумковий залік, який формується з балів, набраних протягом навчання.

## Структура репозиторію

```
bachelor_pythonweb/
├── lesson_01/     # Базова інженерна інфраструктура Python-проєкту
├── lesson_02/     # Основи технології Docker
├── lesson_03/     # Основи Web. Побудова простого HTTP сервера
└── README.md      # Цей файл
```

## Використання

Кожен урок містить:
- `main.py` - основний файл застосунку
- `pyproject.toml` - конфігурація проєкту та залежності
- `README.md` - опис завдання уроку
- `.gitignore` - файли, що ігноруються Git

## Вимоги

- Python 3.12+
- Docker (для уроків з Docker)
- Git

## Встановлення

Для кожного уроку використовується віртуальне середовище:

```bash
# Створення віртуального середовища
uv venv

# Активація (Windows)
.venv\Scripts\activate

# Активація (Linux/Mac)
source .venv/bin/activate

# Встановлення залежностей
uv sync
```
